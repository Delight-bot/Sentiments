# GitHub Safety Checklist ✅

Before pushing to GitHub, verify:

- [x] .env is in .gitignore
- [x] .env is not tracked by git
- [x] Only .env.example is committed
- [ ] Run: `git status` - confirm .env not listed
- [ ] Run: `git add .` then `git status` again - .env should NOT appear
- [ ] Double check: `git diff --cached` - no API keys visible

## If you accidentally commit secrets:
1. IMMEDIATELY revoke the API key at openai.com
2. Use git-filter-repo or BFG Repo-Cleaner to remove from history
3. Force push the cleaned repo
4. Generate new API key
